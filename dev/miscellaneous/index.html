UsefulFunctions._terms — Function
terms(f::Function,Λ::Float64,n::Int64)

Returns the terms for the separable representation of the function f given by f(p,p') = ∑ᵢ gᵢ(p)gᵢ(p') where gᵢ are the terms returned by this function.

Arguments

  • f::Function: The function to be represented.
  • Λ::Float64: Momentum cutoff.
  • n::Int64: Number of terms in approximation to be returned.
